# Shortlistr

AI-powered resume screening system that ranks resumes based on job description relevance using NLP and Machine Learning techniques.

## Tech Stack

* Python
* Scikit-learn
* spaCy
* SentenceTransformers
* Gradio
* Pandas

## Features

* Extracts and parses resume data from PDF files
* Performs skill extraction using NLP
* Calculates resume-job similarity scores
* Ranks candidates based on relevance
* Interactive interface built using Gradio

## Concepts Used

* Natural Language Processing (NLP)
* TF-IDF Vectorization
* Cosine Similarity
* BERT Embeddings
* Machine Learning
